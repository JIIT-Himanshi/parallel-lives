## 🎬 About This Weboreel

**Parallel Lives** is an immersive, one-page cinematic web experience where users explore multiple versions of their life across different realities.

It is designed to feel like a **game + movie combined** — blending interactive storytelling, multiverse navigation, dynamic simulations, and user-driven choices.

> "Every choice creates a new reality."

| Feature | Description |
|---|---|
| 🌌 Cinematic Intro | Engaging opening that sets the multiverse theme |

## 🌐 Vercel Deployment

### Step 1 — Push to GitHub
```bash
git init
git add .
git commit -m "feat: Parallel Lives — multiverse cinematic weboreel"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/parallel-lives.git
git push -u origin main
```

### Step 2 — Deploy on Vercel
1. Go to https://vercel.com → **Add New Project**
2. Import your GitHub repo: `parallel-lives`
3. Use these settings:

| Setting | Value |
|---|---|
| Framework Preset | Other |
| Root Directory | ./ |
| Build Command | (leave empty) |
| Output Directory | (leave empty) |
| Install Command | (leave empty) |

4. Click **Deploy** ✅

## 🎬 About This Weboreel

**Parallel Lives** is an immersive one-page cinematic web experience where users explore multiple versions of their life across different realities. It is designed to feel like a **game + movie combined** — blending interactive storytelling, multiverse navigation, dynamic simulations, and user-driven choices.

> "Every choice creates a new reality."

---

## ✨ Features

- 🌌 Cinematic intro that sets the multiverse theme
- 🌀 Interactive portal system to explore parallel life paths
- 🎭 5 life paths — Artist, Corporate, Entrepreneur, Traveler, Scientist
- 🎮 Choice system — user decisions affect storyline and outcomes
- 📊 Real-time life simulation — tracks happiness, money, freedom, stress
- 🎬 GSAP-powered smooth transitions and effects
- 🌐 Three.js immersive 3D environments
- 🎧 Ambient audio system using Howler.js
- ✨ Glassmorphism + neon futuristic UI
- 📱 Fully responsive for desktop and mobile

## 🎮 How to Experience

1. Open the website and watch the cinematic intro
2. Enter the **Multiverse Hub**
3. Choose a life path — Artist / Corporate / Entrepreneur / Traveler / Scientist
4. Interact and make decisions
5. Watch your life stats change in real-time
6. Explore alternate realities

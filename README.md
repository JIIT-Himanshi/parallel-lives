# 🌌 Parallel Lives — Multiverse Experience

## 🚀 Overview
**Parallel Lives** is a cinematic, interactive web experience that lets users explore multiple versions of their life across different realities.

Unlike traditional websites, this project creates a **game-like immersive environment** where users can interact, make choices, and see how different decisions shape completely different life paths.

---

## 🎯 Key Features

✨ Cinematic intro experience  
🌀 Interactive multiverse hub with portals  
🎭 Multiple life paths:
- 🎨 Artist  
- 🏢 Corporate Professional  
- 🚀 Entrepreneur  
- 🌍 Traveler  
- 🧪 Scientist  

🎮 Real-time interaction system  
📊 Dynamic life simulation (happiness, money, freedom, stress)  
✨ Glassmorphism + Neon UI  
🎬 Smooth animations using GSAP  

---

## 🧠 Concept

> **“What if your life had multiple versions?”**

This project explores parallel realities where each decision leads to a completely different life experience.

---

## 🛠️ Tech Stack

- HTML, CSS, JavaScript  
- GSAP (Animations)  
- Three.js (3D Experience)  
- Lenis (Smooth Scrolling)  
- Howler.js (Audio)
